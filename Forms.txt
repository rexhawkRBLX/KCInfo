**Google Forms**

The following Google Forms can be used for the following things...

**Staff Report Form:**
The Staff Report Form can be used to report any Administrative Staff Member, please answer all questions provided. [Go to link](https://docs.google.com/forms/d/e/1FAIpQLSdwDz2c9h2Cc-chmBgUD7w4covvNh-J-FA6BWonvMLQlPyEOw/viewform?usp=sf_link)

**Ban Request Form:** 
The Ban Request form can be used to request a ban on another member for a reason with provided evidence, please answer all questions provided. [Go to link](https://docs.google.com/forms/d/e/1FAIpQLSd-M5ERj2Y5Tf5nKWjW9P7rU5_irx033m6H-WtybSLqnQYW2Q/viewform?usp=sf_link)

**Ban Appeal (Discord/Game) Form:**
The Ban Appeal form can be used to request a Ban Appeal (to be unbanned) from the game and/or discord... [Go to link](https://docs.google.com/forms/d/e/1FAIpQLSe47MK33CkJKfZNitisQwBvF2pnLzoqkFnRqS7qTBJ8PVxJhw/viewform?usp=sf_link) 

**Suspension Request Form:**
The Suspension Request form can be used to request a Suspension Request on another member... [Go to link](https://docs.google.com/forms/d/e/1FAIpQLSepqYwQzaTE0sBf-Je4M8F_Mg9vCt8C41eZq68_-A9kP9tngQ/viewform?usp=sf_link)
